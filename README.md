# fecli

> 开发 react 组件的脚手架


## 使用

```
$ fe add
$ fe init
$ fe list
$ fe delete
```

## 命令的功能

### fe add

> 全局添加 github 模板

![fe add 例子](./public/add.gif)

### fe list

> 显示模板列表

![fe list 例子](./public/list.gif)

### fe init

> 初始化某一个模板

![fe init 例子](./public/init.gif)

### fe delete

> 删除已添加模板

![fe delete 例子](./public/delete.gif)
